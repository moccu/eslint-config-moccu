# eslint-config-moccu

[![Build Status](https://travis-ci.org/moccu/eslint-config-moccu.svg?branch=master)](https://travis-ci.org/moccu/eslint-config-moccu)

The eslint config we use at moccu

## Installation

```
$ npm install --save-dev eslint-config-moccu
```

## Usage

```js
{
  "extends": "moccu",
  "rules": {
    // Additional rules...
  }
}
```

### React

```js
{
  "extends": "moccu/react"
}
```
